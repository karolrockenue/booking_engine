C · The pages

What changes inside the content area for each sidebar item. Distinctive layout per page so you can recognise them at a glance.

◉Overview /[id]
KPI grid + recent bookings
Top: stat tiles (bookings · 7d, revenue · 7d, occupancy, ADR). Below: latest 5 bookings list + quick links.
▤Bookings /[id]/bookings
Sortable table · 200-row cap
Filter by status / rate type / date. Each row → side panel with full booking detail, extras, payment events, cancel action.
¶Content /[id]/content
5 content blocks · live site copy
Edit Hero / Neighbourhood / Good to know / Contact / Footer. Some fields auto-fill from Cloudbeds (labelled).
▣Media /[id]/media
R2 photo library · 3 variants per file
Upload, slot (Hero / Gallery / Room / Neighbourhood / Marketing), reorder, alt text. Photos appear in the Emails composer too.
≡Rate plans /[id]/rates
Mirrored from Cloudbeds · editable policy
Read-only: name, room type, price ladder. Editable: refundable Y/N + cancellation deadline / penalty.
✉Emails /[id]/emails
5 templates · composer · schedules · send log
List of 5 templates → per-template Unlayer composer (drag/drop blocks, brand fonts, library picker). Sub-pages: schedule rules + send log.
⇄Alerts /[id]/alerts
Stub · alerts engine pending
Operational signals derived from payment_events + cron heartbeats: missed crons, repeated auto-charge failures, auto-cancels.
●Cloudbeds /[id]/cloudbeds
OAuth status · sync · webhooks
Connect / disconnect, last sync time, manual "Sync now" trigger, webhook subscriptions list, scopes granted.
$Stripe /[id]/stripe
Connect account · payouts · refunds
Account status pill, currency, balance, recent payouts, refunds. "Manage in Stripe" link to their dashboard.
⌘Domain & deploy /[id]/domain
Stub · DNS / SSL / Railway probe
Pending. Will show domain record status, SSL cert state, last Railway deploy + theme env var.

Composer rules

  1. Drag a block, library auto-opens Drop an Image block → our Media library appears so you pick a photo. No "select" hunt needed.
  2. Upload-from-disk lands in Media Files dropped on an Image block upload to R2 + create a row in the Media library (slot = Marketing).
  3. Variables via {{…}} Type { in any text block to autocomplete {{guest.firstName}} etc.
  4. Brand fonts at the top of the list Cormorant Garamond + Inter for Portico; system fonts (Arial, Georgia…) below.
  5. Save = render + cache Save exports Unlayer's HTML and stores it on the row. Real sends substitute {{…}} into that cached HTML — no Unlayer needed server-side.
