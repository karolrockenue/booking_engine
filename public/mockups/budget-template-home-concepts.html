Budget template — homepage concepts

Five variations on the traditional hotel pattern: big London hero, header + nav, availability bar. Same hotel placeholder (Kings Cross Lodge) across all five so you're comparing design choices, not content. Each variation moves one or two knobs — header treatment (white / transparent / centred), search-bar placement (overlapping hero / inside hero / below hero), accent colour, headline alignment, type voice (serif / sans / cinematic). Stock imagery via picsum (London-ish).

M1

White header · serif headline · overlapping search

The baseline. White header above; full-bleed London hero with a centred serif headline; gold accent; availability bar overlaps the bottom edge of the hero into the white below. Premier Inn / Hilton / every chain hotel does this for a reason.

M2

Transparent header · bottom-left copy · search bar below

M3

Centred logo · classical British · burgundy

M4

Modern sans · search inside hero · navy

Bigger departure: drop the serif for a confident sans display headline, align everything left, put the availability bar inside the hero (not overlapping out). Navy accent instead of gold. Feels like a newly-launched London hotel — Yotel-ish ambition without the cyber-bunker aesthetic. Could read younger / more modern, though some hotels might feel it's "not hotel-y enough."
