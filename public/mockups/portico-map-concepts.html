01

OSM dark filter.

What ships now. Standard OpenStreetMap tiles re-coloured client-side via filter: invert(0.92) hue-rotate(180deg) saturate(0.6). Free, no API key, instant. Looks washed-out on the dark theme; labels go faintly readable.

02

Carto Positron.

Carto's clean light style — pale parks, near-white roads, restrained labels. Free for non-commercial via basemaps.cartocdn.com. Looks excellent on the Ivory theme; on Dark you'd invert it (concept 03 instead). No API key.

03

Carto Dark Matter.

Carto's true dark style — purpose-built dark cartography, no filter hacks. Slate roads, near-black ground, subtle labels. Free for non-commercial. Best fit for the Dark theme — sits next to the wardrobe-doors photo without the uncanny-filter feeling.

04

Illustrated SVG.

No map tiles at all — a hand-drawn schematic of just the bits that matter (Hyde Park as a wash, Paddington as a station glyph, the hotel as a brass pin). Pure brand. Static, no live data, but loads instantly and reads as a postcard. Can be redrawn properly by a designer.

05

Typography only.

No map at all — a typographic location card. Big serif address, GPS coordinates, three travel routes with timings. Most editorial; reads like a printed reservation card. The "Open in maps →" pill keeps the actual map one click away.
